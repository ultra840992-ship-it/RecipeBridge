## Bitz의 매칭/결제 라우팅 코드 보안 취약점 분석 요청 반려

**대상 에이전트:** Bitz

**요청 태스크:** Bitz가 작성할 매칭/결제 라우팅 코드 구조의 SQL 인젝션 및 보안 취약점 사전 모의 해킹 분석 보고서 작성.

**반려 사유:**
요청된 분석 작업을 수행하기 위한 **Bitz의 매칭/결제 라우팅 코드 파일이 현재 존재하지 않습니다.** Verity의 임무는 타 에이전트의 산출물(코드, 문서 등)을 직접 검증하여 할루시네이션 및 논리적 오류를 통제하는 것입니다. 실체가 없는 코드에 대한 사전 분석은 불가능하며, 이는 Verity의 업무 수행 범위에 해당하지 않습니다.

**조치 요구:**
Bitz는 매칭/결제 라우팅 코드 작성을 완료하고 해당 파일을 제출해야 합니다. 코드 제출 후, Verity는 `view_file` 도구를 사용하여 실제 코드를 검토하고 보안 취약점 분석을 진행할 것입니다.

**무결성 원칙:**
진실이 아니면 통과할 수 없습니다. 검증할 산출물이 없으므로, 해당 태스크는 현재 상태로 처리할 수 없습니다.
