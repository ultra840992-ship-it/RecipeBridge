## 태스크 반려 알림: Bitz 에이전트의 코드 구조 사전 분석 보고서 작성

**담당 에이전트**: Verity (베리티)
**지시 태스크**: [Lv2] Bitz가 작성할 매칭/결제 라우팅 코드 구조의 SQL 인젝션 및 보안 취약점 사전 모의 해킹 분석 보고서 작성

**검토 결과**:
해당 태스크는 Verity의 업무 수행 범위와 절차에 부합하지 않아 반려 처리됩니다.

**상세 사유**:
1.  **업무 범위 위반**: Verity는 마스터 Aegis가 주재하거나 타 에이전트가 **완료한 결과물(코드 파일, 마크다운 문서 등)**을 직접 읽어보고 논리적 결함이나 지시 불이행 여부를 교차 검증하는 역할을 수행합니다. 현재 지시된 태스크는 'Bitz가 작성할' 코드에 대한 '사전 모의 해킹 분석 보고서 작성'을 요구하며, 이는 **미완료된 작업물**에 대한 **새로운 분석 보고서 생성**에 해당합니다.
2.  **도구 사용 불가**: Verity는 `view_file`, `grep_search` 툴을 통해 **기존에 존재하는 파일**을 직접 확인하여 검증합니다. 현재는 Bitz 에이전트의 코드 산출물이 존재하지 않으므로, Verity의 핵심 검증 도구를 사용할 수 없습니다.
3.  **역할 제약 위반**: Verity의 역할은 '코드를 직접 짜거나 기획을 수정하지 마십시오. 오직 비판(Critic)과 반려(Reject)만이 당신의 역할입니다.'라고 명확히 규정되어 있습니다. '사전 모의 해킹 분석 보고서 작성'은 새로운 산출물을 생성하는 행위로, Verity의 본래 역할인 기존 산출물에 대한 비판 및 반려 범주를 벗어납니다.

**결론**:
Bitz 에이전트의 매칭/결제 라우팅 코드 작성이 완료된 후, 해당 코드 파일에 대한 검증 요청 시에만 Verity의 업무 수행이 가능합니다. 미완료된 작업물에 대한 사전 분석 보고서 작성은 Verity의 임무가 아닙니다.
