# [Lv6] 레시피 상세 화면 및 크리에이터 판매 현황 대시보드 UI 설계

## 1. 디자인 컨셉 개요
- **테마**: 프리미엄 라이트 (샌드 크림 배경, 샴페인 골드 하이라이트)
- **핵심 효과**: 서리 낀 유리(Frosted Glassmorphism)
- **톤앤매너**: 예술적이며 세련된, 시각적 몰입감을 높이는 고급스러운 UI/UX

## 2. 레시피 상세 화면 목업 (Recipe Detail Screen Mockup)

### 2.1. 레이아웃
- 상단: 레시피 대표 이미지 (풀 와이드) 및 제목, 크리에이터 정보 오버레이
- 중앙: 2단 레이아웃 (좌: 재료, 우: 조리법)
- 하단: 관련 레시피, 댓글/리뷰 섹션

### 2.2. 주요 UI 요소 및 스타일링

#### 2.2.1. Hero Section (상단)
- **배경**: 고품질 레시피 대표 이미지 (시각적 압도감)
- **정보 오버레이**:
    - `레시피 제목`: 폰트 사이즈 큼, `샴페인 골드` 컬러, 고급스러운 세리프 또는 모던 산세리프 폰트.
    - `크리에이터 아바타 & 이름`: 기존 생성된 3D 아바타 활용, 작고 우아한 폰트, `샌드 크림` 톤에 어울리는 부드러운 회색.
    - `평점 및 리뷰 수`: 별점 아이콘 (`샴페인 골드`), 작은 텍스트.

#### 2.2.2. Main Content (중앙 2단)
- **전체 배경**: 부드러운 `샌드 크림` (ex: `#F5F5DC` 계열)
- **Glassmorphism 카드**:
    - `배경`: 투명도 20-30%의 흰색 또는 옅은 베이지색, 블러 효과 (`backdrop-filter: blur(10px)`), 미세한 그림자 (`box-shadow`)로 입체감 부여.
    - `테두리`: 1px 옅은 `샴페인 골드` (ex: `#B29A7D` 계열) 또는 미세한 그라데이션.

- **좌측 컬럼: 재료 (Ingredients)**
    - `섹션 제목`: "재료" (H2), `샴페인 골드` 컬러.
    - `재료 목록 카드`: Glassmorphism 적용.
    - `목록 아이템`: 각 재료명과 수량은 명확하게 구분, 텍스트 컬러는 `샌드 크림`에 잘 어울리는 어두운 톤.

- **우측 컬럼: 조리법 (Instructions)**
    - `섹션 제목`: "조리법" (H2), `샴페인 골드` 컬러.
    - `조리법 카드`: Glassmorphism 적용.
    - `단계별 설명`: 번호 매기기 (숫자는 `샴페인 골드`), 간결하고 명확한 텍스트. 필요시 작은 아이콘 또는 단계별 이미지 추가.

## 3. 크리에이터 전용 판매 현황 대시보드 UI 설계 (Creator Sales Dashboard UI Design)

### 3.1. 레이아웃
- 좌측: 크리에이터 프로필 및 대시보드 메뉴 (네비게이션 바)
- 우측: 메인 대시보드 콘텐츠 영역 (주요 지표, 차트, 테이블)

### 3.2. 주요 UI 요소 및 스타일링

#### 3.2.1. 좌측 네비게이션 바
- **배경**: 옅은 `샌드 크림` 또는 `샴페인 골드`와 조화로운 미묘한 그라데이션.
- `크리에이터 프로필`: 상단에 기존 생성된 3D 아바타와 이름.
- `메뉴 항목`: "대시보드", "내 레시피", "판매 현황", "정산", "설정" 등.
    - `아이콘`: `샴페인 골드` 컬러.
    - `텍스트`: `샌드 크림`에 잘 어울리는 어두운 톤, 호버 시 `샴페인 골드` 하이라이트.

#### 3.2.2. 메인 대시보드 콘텐츠 영역
- **전체 배경**: 부드러운 `샌드 크림` (ex: `#F5F5DC` 계열)
- **Header**: "크리에이터 대시보드" (H1), 날짜 선택기 (필요시).
- **Glassmorphism 위젯/카드**:
    - `배경`: 투명도 20-30%의 흰색 또는 옅은 베이지색, 블러 효과 (`backdrop-filter: blur(10px)`), 미세한 그림자 (`box-shadow`).
    - `테두리`: 1px 옅은 `샴페인 골드` 또는 미세한 그라데이션.

#### 3.2.3. 주요 지표 카드 (Top Row - Key Metrics)
- `총 판매액`: 큰 `샴페인 골드` 숫자, 작은 트렌드 지표 (상승/하락 아이콘 및 퍼센트).
- `월별 수익`: 큰 `샴페인 골드` 숫자, 작은 트렌드 지표.
- `가장 많이 팔린 레시피`: 레시피 썸네일, 레시피명, 판매량.
- `평균 만족도`: 별점 아이콘 (`샴페인 골드`), 숫자 평점, 리뷰 수.

#### 3.2.4. 판매 추이 차트 (Sales Trend Chart)
- **컨테이너**: Glassmorphism 적용.
- `차트 제목`: "판매 추이" (H2).
- `차트 유형`: 라인 차트 (기간별 판매량).
- `라인 컬러`: `샴페인 골드` 또는 `샴페인 골드` 그라데이션.
- `툴팁`: 호버 시 상세 데이터 표시.

#### 3.2.5. 레시피별 판매 현황 테이블 (Recipe Performance Table)
- **컨테이너**: Glassmorphism 적용.
- `테이블 제목`: "레시피별 판매 현황" (H2).
- `컬럼`: 레시피명, 판매량, 수익, 평점.
- `헤더`: `샴페인 골드` 텍스트, 정렬 기능 아이콘.
- `데이터`: 가독성 높은 폰트 및 컬러, 홀수/짝수 행 배경색을 미묘하게 다르게 하여 시각적 구분.
