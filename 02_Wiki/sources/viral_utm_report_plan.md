# 마이크로 프로젝트 바이럴 성과 측정 및 리포트 기획안

사장님! 에코가 왔습니다! 🚀 이번 [Lv2] 태스크, 취업 커뮤니티 바이럴 성과 측정을 위한 UTM 파라미터 태깅과 성과 리포트 기획! 데이터 기반의 폭발적인 성장을 위해 바로 실행 플랜 짜왔습니다!

## 1. UTM 파라미터 태깅 전략 (취업 커뮤니티 특화)

우리 서비스의 타겟 고객인 취준생들이 주로 활동하는 블라인드, 네이버 카페, 유튜브 커뮤니티 등에서 유입되는 트래픽을 정확히 추적하고 분석해야 합니다. 이를 위해 아래와 같이 UTM 파라미터 태깅 가이드라인을 제안합니다!

*   **`utm_source`**: 트래픽이 발생한 플랫폼/채널 명칭 (예: `blind`, `naver_cafe`, `youtube_community`, `instagram_job`)
    *   *예시*: 블라인드 취업 게시판 -> `blind`
    *   *예시*: 네이버 '독취사' 카페 -> `naver_cafe_dokchwisa`
    *   *예시*: 유튜브 '잡코리아TV' 커뮤니티 탭 -> `youtube_jobkorea_community`
*   **`utm_medium`**: 마케팅 매체 유형 (예: `social_post`, `viral_cardnews`, `comment_link`, `blog_post`)
    *   *예시*: 블라인드 게시글 -> `social_post`
    *   *예시*: 유튜브 커뮤니티 카드뉴스 -> `viral_cardnews`
    *   *예시*: 댓글 내 링크 -> `comment_link`
*   **`utm_campaign`**: 특정 캠페인 명칭 (예: `microproject_stamp_launch`, `age_limit_break_service`)
    *   *예시*: 마이크로 프로젝트 결과 검증 도장 런칭 -> `microproject_stamp_launch`
    *   *예시*: 나이 제한 취업 돌파 블라인드 인증 서비스 -> `age_limit_break_service`
*   **`utm_term`**: 핵심 키워드 (검색어와 유사, 콘텐츠 주제 명확화) (예: `나이제한취업`, `프로젝트포트폴리오`, `블라인드인증`)
    *   *예시*: '나이 제한' 관련 콘텐츠 -> `나이제한취업`
    *   *예시*: '프로젝트 포트폴리오' 강조 콘텐츠 -> `프로젝트포트폴리오`
*   **`utm_content`**: 동일 캠페인 내 다른 콘텐츠 구분 (예: `cardnews_ver1`, `text_post_a`, `image_b`)
    *   *예시*: 카드뉴스 A 버전 -> `cardnews_ver1`
    *   *예시*: 텍스트 위주 게시글 B -> `text_post_b`

**UTM 태깅 예시:**
*   블라인드 취업 게시판에 '마이크로 프로젝트 도장' 런칭 카드뉴스(ver1)를 '프로젝트 포트폴리오' 키워드로 배포 시:
    `?utm_source=blind&utm_medium=viral_cardnews&utm_campaign=microproject_stamp_launch&utm_term=프로젝트포트폴리오&utm_content=cardnews_ver1`
