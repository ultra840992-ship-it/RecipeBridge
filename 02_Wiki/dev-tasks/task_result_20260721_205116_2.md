- **모달 라우팅 연동 확인**: 매칭 상태 변경 시, 특정 모달이 뜨고 브라우저 URL 라우팅이 동시에 변경되는지 확인.
- **테스트 방법**:
    1.  `run_dashboard.bat`을 가동해 프론트엔드 대시보드를 실행한다.
    2.  백엔드 API를 통해 매칭 상태를 `'matched'`로 변경한다. (예: `curl -X POST -H "Content-Type: application/json" -d '{"status": "matched"}' http://localhost:5000/api/match_status`)
    3.  프론트엔드에서 모달이 나타나고 URL이 `/match-found` 등으로 변경되는지 육안으로 확인한다.
    4.  매칭 상태를 `'pending'` 등으로 다시 변경했을 때, 모달이 닫히고 URL이 초기 상태로 돌아오는지 확인한다.

- 프론트엔드 코드 수정 시: `validate-template.sh index.js`
