# RecipeBridge MVP 런칭을 위한 에이전트별 Action List 및 4주 로드맵

이 문서는 RecipeBridge 플랫폼의 신속한 MVP(최소 기능 제품) 런칭을 위해 6인의 가상 에이전트가 수행해야 할 상세 업무 액션 리스트와 주차별 목표 일정입니다.

---

## 1. 에이전트별 상세 Action List (초기 가동 용)

### 👤 Aegis (이지스) | 마스터 에이전트 (공신력 확보 및 인증 보증)
- **Pain Point 타격**: 중고 신입을 선호하는 기업들의 불신을 격파하기 위해, 구직자가 제출한 포트폴리오의 **'실제 기여도와 신뢰성'**을 AI 마스터 엔진으로 철저하게 무조건 검증(Proof of Contribution).
- **Action Item**:
  - [x] **[Critical]** 구직자가 제출한 마이크로 프로젝트 산출물 코드에 대한 '표절률 및 AI 대필 진위 여부' 자동 검증 프롬프트 설계.
  - [x] **[Critical]** 프로젝트 코드의 정상 작동 및 빌드 여부를 동적으로 체크하여 경력 인증서 발급 승인 도장(Secret Chain Sign) 검수.
  - [x] 주간 마일스톤 진척도 모니터링 및 병목 해결.
  - [x] 개발팀 Bitz의 완성된 PR 코드 검수 및 배포 승인 (오작동/유실 교차 검증).

### 👤 Nova (노바) | 기획 에이전트 (나이 제한 무관 블라인드 매칭)
- **Pain Point 타격**: 이력서 필터링으로 30대 중반 이상의 쌩신입들이 서류 탈락하는 **'나이 제한 장벽'**을 파괴하기 위해, 학벌과 나이가 아예 표기되지 않는 **'블라인드 실무 역량 인증 매칭 명세'** 기획.
- **Action Item**:
  - [x] **[Critical]** 나이/학벌 필드 자체가 없는 '블라인드 프로젝트 기여 인증 프로필 UI' 및 매칭 프로세스 기획.
  - [x] **[Critical]** 스타트업들이 실제로 현업에서 골머리를 썩고 있는 1~2주용 단기 마이크로 프로젝트 과제 스펙 세부 기획 (예: SEO 최적화 패치, 이탈율 분석 모듈 등).
  - [x] 스타트업 대상 단기 과제 모집 양식 및 가이드 명세서 템플릿 기획.
  - [ ] 구직자용 AI 레시피 스토어 판매용 '기본 웹 빌드 레시피 팩' 3종 프롬프트 구성.
  - [ ] 마이크로 프로젝트 진행 및 결과물 경력 인증서(인증 체인) 프로세스 상세 설계.

### 👤 Vivid (비비드) | 디자인 에이전트
- **Action Item**:
  - [x] 피그마를 활용한 RecipeBridge 메인 화면, 프로젝트 리스트, 레시피 스토어 와이어프레임 설계.
  - [x] 프리미엄 다크 모드 톤앤매너, 폰트(Outfit, Inter), HSL 기반 컬러 토큰 정의.
  - [x] 반응형 레이아웃 및 UX 애니메이션 가이드라인을 개발팀 Bitz에 공급.

  - [x] 구글 서치 콘솔 연동 및 색인 등록 자동화 루틴 테스트.

  - [x] 의뢰 스타트업과 구직자 매칭 실패 시 불만율 피드백 수집 채널 개설.

## 2. 4일 초단기 목표 일정 (타임라인)

```
[Day 1: 기획 & 디자인 완비] ──> [Day 2: 핵심 기능 개발] ──> [Day 3: 인프라 & 마케팅] ──> [Day 4: 검증 & 정식 오픈]
```

### 📅 Day 1: 기획 구체화 및 디자인 토큰 확정
- **목표**: 서비스 명세서 완비 및 UI/UX 테마 가이드 배포.
- **주요 산출물**: 
  - Nova: 스타트업 과제 양식서, 경력 인증 시나리오 기획.
  - Vivid: 피그마 와이어프레임 완성, HSL 컬러 및 타이포 CSS 변수 배포.
  - Aegis: Day 1 기획/디자인 산출물 직접 파일 검수 및 승인.

### 📅 Day 2: 핵심 프론트 및 DB 연동 개발
- **목표**: React 기반 웹 서비스 퍼블리싱 및 노션 DB 연동 완성.
- **주요 산출물**:
  - Bitz: UI 퍼블리싱 완료, `src/` 폴더 내 소스코드 빌드, Notion API 연동.
  - Nova: 3가지 기본 레시피 팩 프롬프트 패키징 완료.
  - Carey: GA4 분석 태그 및 이탈율 수집 설계서 작성.

### 📅 Day 3: 마케팅 자동화 및 인프라 구축
- **목표**: 텔레그램 실시간 제어 구축, VPS 상시 가동 셋팅, 마케팅 크론 작동.
- **주요 산출물**:
  - Echo: 사전 예약용 랜딩 페이지 SEO 색인 수동 등록 완료, 카드뉴스 자동 크론 구동.
  - Bitz/Aegis: 백업/복원 스크립트 고도화 및 텔레그램 연동 봇 백그라운드 안정화.

### 📅 Day 4: 종합 QA 검수 및 베타 오픈
- **목표**: 전 시스템 정합성 체크 및 정식 서비스 오픈.
- **주요 산출물**:
  - Carey: 베타 테스터 50인 고객 만족도 및 고통 보고서 수집 파이프라인.
  - Aegis: 2단계 거짓말 검수 가이드를 통해 전체 `src/` 소스 코드 QA 점검 및 승인.
  - **결과**: `RecipeBridge v1.0.0` 상용 서버 정식 배포 오픈.
