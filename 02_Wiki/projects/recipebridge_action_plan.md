# RecipeBridge MVP 런칭을 위한 에이전트별 Action List 및 4주 로드맵

이 문서는 RecipeBridge 플랫폼의 신속한 MVP(최소 기능 제품) 런칭을 위해 6인의 가상 에이전트가 수행해야 할 상세 업무 액션 리스트와 주차별 목표 일정입니다.

---

## 1. 에이전트별 상세 Action List (초기 가동 용)

### 👤 Orchestrator (Aegis) | 오케스트레이터 (R&R 및 KPI 총괄 제어)
- **Pain Point 타격**: 중고 신입을 선호하는 기업들의 불신을 격파하기 위해, 구직자가 제출한 포트폴리오의 **'실제 기여도와 신뢰성'**을 AI 마스터 엔진으로 철저하게 무조건 검증(Proof of Contribution).
- **Action Item**:
  - [x] **[Critical]** 구직자가 제출한 마이크로 프로젝트 산출물 코드에 대한 '표절률 및 AI 대필 진위 여부' 자동 검증 프롬프트 설계.
  - [x] **[Critical]** 프로젝트 코드의 정상 작동 및 빌드 여부를 동적으로 체크하여 경력 인증서 발급 승인 도장(Secret Chain Sign) 검수.
  - [x] 주간 마일스톤 진척도 모니터링 및 병목 해결.
  - [x] 개발팀 Bitz의 완성된 PR 코드 검수 및 배포 승인 (오작동/유실 교차 검증).
  - [x] **[Advance]** a-알파 런칭을 위한 전체 과제 DB 정합성 최종 검수 및 스타트업 승인 프로세스 마스터 정책 수립.
  - [x] **[Critical]** [Lv2] 결제 및 매칭에 따른 구직자 기여도 인증서 스마트 계약/해시 서명 체계 검증 및 최종 보안 정책 수립.
  - [x] **[Critical]** [Lv3] 베타 서비스 출시 대비 에이전트 6인의 최종 릴리즈 본(코드/마크다운) 정합성 교차 QA 및 보안 서명 배포 승인.
  - [x] **[Critical]** [Lv4] 정식 런칭 후 실시간 트래픽 대응 비즈니스 모니터링 및 AI 오작동 롤백 시스템 최종 배포.
  - [x] **[Critical]** [Lv5] 글로벌 매칭 모델 및 Stripe 다중 외화 결제 정책 검수 및 승인.
  - [x] **[Critical]** [Lv6] C2C 거래 규제 대응 및 정산금 분쟁 중재 마스터 정책 승인.
  - [x] **[Critical]** [Lv7] ATS 연동 데이터 무결성 검수 및 구직자 기여도 인증 최종 승인.
  - [ ] **[Critical]** [Lv8] 24/7 장애 에스컬레이션 롤백 시스템 검수 및 정식 상용화 마스터 배포 승인.

  - [x] **[Advance]** 중고 신입-스타트업 블라인드 매칭 수수료 과금 모델(BM) 기획 명세서 및 결제 시나리오 초안 작성.
  - [x] **[Critical]** [Lv2] 결제 모듈 도입에 따른 수수료 차등 모델 및 매칭 탈락 시 구직자 보상 환불 정책(시나리오) 확정.
  - [x] **[Critical]** [Lv3] '블라인드 프로젝트 기여 인증 명세서'를 정식 채용 연계에 활용하기 위한 채용 대행 연계 프로세스 최종 정책서 수립.
  - [x] **[Critical]** [Lv4] 베타 테스터 50인 피드백 데이터 기반 2차 기획 및 B2B 채용 연계 마이크로 단기 과제 확장 기획서 수립.
  - [x] **[Critical]** [Lv5] 글로벌 아웃소싱 표준 과제 양식 기획 및 영문 블라인드 채용 프로세스 정립.
  - [x] **[Critical]** [Lv6] 레시피 스토어 거래 수수료 모델 기획 및 크리에이터 보상 정책서 작성.
  - [x] **[Critical]** [Lv7] B2B 타겟 단기 인턴십 기여도 검증 및 채용 연계 ATS 시스템 시나리오 기획.
  - [ ] **[Critical]** [Lv8] 1인 기업 AX(Agentic Transformation) 완전 자동화 운영 예외 대응 정책서 수립.
